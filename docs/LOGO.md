# Logo

## Concept

Logo 由三部分组成：

- 圆角蓝绿色底：表示浏览器扩展和工具属性
- 白色对话气泡：表示网页文本和翻译上下文
- 中英视觉符号：表示双语和跨语言转换

## Files

- `assets/logo.svg`：完整 logo，适合 README、网站和大尺寸展示
- `assets/logo-mark.svg`：紧凑 mark，适合图标和头像
- `assets/icons/icon-16.png`
- `assets/icons/icon-32.png`
- `assets/icons/icon-48.png`
- `assets/icons/icon-128.png`

## Usage

图标已经在 `manifest.json` 的 `icons` 和 `action.default_icon` 中配置。
